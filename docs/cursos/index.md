# Cursos

Tutoriais reutilizáveis sobre tópicos centrais de bioinformática. Cada curso é independente e pode ser feito por conta própria, ou como parte de um dos [eventos](../eventos/index.md) do programa.

<div class="grid cards" markdown>

-   :material-console: **Bash / Linux para bioinformática**

    ---

    Linha de comando, manipulação de arquivos de sequências, pipes e scripts básicos para o dia a dia em bioinfo.

    **Carga horária:** ~4h
    **Pré-requisitos:** Nenhum

    :octicons-check-circle-fill-24:{ .green } **Disponível**

    [:octicons-arrow-right-24: Acessar curso](bash/index.md)

-   :material-magnify-scan: **BLAST**

    ---

    Busca por similaridade entre sequências: do conceito ao uso na linha de comando e na interpretação de resultados tabulares.

    **Carga horária:** ~4h
    **Pré-requisitos:** Bash básico

    :octicons-clock-fill-24:{ .yellow } **Em construção**

    [:octicons-arrow-right-24: Acessar curso](blast/index.md)

-   :material-graph-outline: **Filogenia molecular**

    ---

    Inferência evolutiva com dados moleculares: alinhamento (MAFFT/MUSCLE/pal2nal), filtragem (trimAL), concatenação, IQ-TREE 3 com partições, ModelFinder e visualização no iTOL.

    **Carga horária:** ~6h
    **Pré-requisitos:** Bash básico, biologia molecular

    :octicons-check-circle-fill-24:{ .green } **Disponível**

    [:octicons-arrow-right-24: Acessar curso](filogenia/index.md)

    :octicons-check-circle-fill-24:{ .green } **Disponível**

## Como os cursos estão organizados

Todos seguem a mesma estrutura, pra você sempre saber o que esperar:

- :material-target: **Objetivos de aprendizagem** — o que você vai saber fazer ao final
- :material-clock-outline: **Carga horária estimada** — tempo aproximado para concluir o material
- :material-tools: **Pré-requisitos** — softwares, conhecimentos prévios e dados de exemplo
- :material-script-text-outline: **Conteúdo modular** — dividido em etapas curtas, cada uma autocontida
- :material-pencil-outline: **Exercícios** — para fixar o aprendizado, com gabarito dobrável
- :material-folder-download-outline: **Materiais** — datasets de exemplo no próprio repositório, prontos para baixar
