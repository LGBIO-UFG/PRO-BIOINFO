# Cursos

Tutoriais reutilizáveis sobre tópicos centrais de bioinformática. Cada curso é independente e pode ser feito por conta própria, ou como parte de um dos [eventos](../eventos/index.md) do programa.

## Como os cursos estão organizados

Todos seguem a mesma estrutura, pra você sempre saber o que esperar:

- :material-target: **Objetivos de aprendizagem** — o que você vai saber fazer ao final
- :material-clock-outline: **Carga horária estimada** — tempo aproximado para concluir o material
- :material-tools: **Pré-requisitos** — softwares, conhecimentos prévios e dados de exemplo
- :material-format-list-numbered: **Conteúdo modular** — dividido em etapas curtas, cada uma autocontida
- :material-pencil-outline: **Exercícios** — para fixar o aprendizado, com gabarito dobrável
- :material-folder-download: **Materiais** — datasets de exemplo no próprio repositório, prontos para baixar
